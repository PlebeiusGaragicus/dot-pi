# playwright-cli Installation

## Prerequisites

- **Node.js** >= 18

## Install

```bash
npm install -g @playwright/cli
```

## Install Browser Binaries

```bash
npx playwright install
```

This downloads Chromium, Firefox, and WebKit binaries that Playwright needs.

## Verify

```bash
playwright-cli --version
```
